# AI Tools Adoption & Analytics Dashboard

## Project Overview

This Power BI dashboard analyzes global AI tools adoption trends, usage insights, and yearly growth patterns.

The dashboard provides interactive visualizations to understand how AI tools are being adopted across different countries and platforms.

---

## Features

* AI tool adoption comparison
* Country-wise adoption analysis
* Yearly growth trends
* Interactive Power BI visuals
* Clean dashboard design

---

## Tools Used

* Power BI
* Data Visualization
* Data Analysis
* Microsoft Excel

---

## Dashboard Insights

* ChatGPT shows the highest adoption rate among AI tools.
* AI adoption trends increased significantly over time.
* Multiple countries demonstrated balanced AI tool usage patterns.
